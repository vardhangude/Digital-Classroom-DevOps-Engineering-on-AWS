## 📖 Overview

This repository documents my hands-on learning journey through the **DevOps Engineering on AWS** Digital Classroom. Throughout this course, I explore modern DevOps practices, automation techniques, and cloud-native development workflows using AWS services.

The focus is on implementing Infrastructure as Code (IaC), Continuous Integration and Continuous Delivery (CI/CD), deployment automation, monitoring, and collaboration using AWS-native tools and industry best practices.

---

## 🎯 Learning Objectives

Throughout this course, I am gaining practical experience in:

- Designing and implementing DevOps workflows on AWS
- Building Continuous Integration (CI) pipelines
- Implementing Continuous Delivery (CD) practices
- Automating infrastructure using Infrastructure as Code (IaC)
- Developing secure and scalable deployment strategies
- Managing source code using Git-based workflows
- Building automated build, test, and deployment pipelines
- Deploying applications to multiple AWS compute services
- Monitoring applications and cloud infrastructure
- Applying DevOps best practices for collaboration and automation
